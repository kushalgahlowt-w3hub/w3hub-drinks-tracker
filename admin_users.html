User Management

Link Supabase Auth users to roles (admin / runner). Only users listed here can access the app.

How to add a new user

  1. In Supabase Dashboard → Authentication → Users, create a new user with their email.
  2. Copy that user’s UUID (their id in the Users table).
  3. Paste the UUID and email below, choose a role, and click Add User Mapping.
  4. The user can then go to the login page and use “Forgot password?” to set a password.
